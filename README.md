How to Install
What You Need
A web server (like XAMPP, WAMP, or hosting service)

PHP 7.4 or newer

MySQL database

Setup Steps
Upload Files

Put all the PHP files in your website folder

Setup Database

Create a MySQL database named akaripic

The website will create all needed tables automatically

Configure Database

Open the PHP file and find these lines:

php
$servername = "localhost";
$username = "root";  // Your database username
$password = "";      // Your database password
$dbname = "akaripic";
Change the username and password to match your database

Test the Website

Open your website in a browser

You should see the AnimeGIF homepage

Troubleshooting
Common Problems
Can't connect to database

Check your database username and password

Make sure MySQL is running

Verify the database name is akaripic

Pictures not loading

Check your internet connection

The Nekos.best API might be temporarily down

Login problems

Make sure you're using a Gmail address

Check that your password is correct

Try resetting your password

Support
If you have problems:
